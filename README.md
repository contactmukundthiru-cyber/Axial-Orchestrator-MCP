# AXIAL v1 (MAX)

Local-first cognitive command center.

## Project Structure

- `crates/`: Rust workspace core components
- `apps/`: UI and IDE extensions
- `scripts/`: Development and maintenance scripts
- `docs/`: Architecture and design documentation

## Dev Environment

Use VS Code Dev Containers for the recommended development environment.
- Profile A: `base`
- Profile B: `docker`
- Profile C: `gpu`

## Phase 1: Foundational CLI

Run `axial doctor` to verify your environment.
